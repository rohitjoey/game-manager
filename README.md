# Football Game Manager

## Project Overview
A simple program that keeps track of the score during the game, stores the final result after the game ends, and displays some statistics

---

## Tech Stack
- **Node.js**  
- **Express.js**  
- **Prisma ORM**  
- **PostgreSQL**  
- **TypeScript**
